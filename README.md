# Flash Cards, React Native - Quiz

 - Android
Project developed for the purpose of react native studies, offered by udacity


## Getting Started &#x2705;

To get started app:

* Clone the project
* install all project dependencies with `npm install`
* start the development server with `npm start || init expo`

## How Use

* Open your android studio and run
* Using Nexus 4 API 28
